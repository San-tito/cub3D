## 🚀 Getting Started

### 🔧 Installation

1. Clone the Minishell repository:
```sh
git clone --recursive https://github.com/San-tito/cub3D
```

2. Change to the project directory:
```sh
cd cub3D
```

3. Compile the dependencies:
```sh
make
```

### 🤖 Running cub3D
```sh
./cub3d
```
